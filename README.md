# blockish-caca

video players in the terminal with blockish over libcaca with LD_PRELOAD magic

# building it

```shell
$ cargo build --release
```

# using it (rust)

```shell
$ cargo run mplayer path/to/your/video
```

```shell
$ cargo run mpv path/to/your/video
```

# using it (shell)

```shell
$ ./mplayer path/to/your/video
```

```shell
$ ./mpv path/to/your/video
```
